# ************************** #
0、安装java并配置环境，运行关闭快速编辑.bat（运行后通过在cmd双击右键选择标记可以选择内容，然后title处右键编辑复制可复制内容）
1、将tika文件夹下的两个文件复制粘贴到C:\Users\用户名\AppData\Local\Temp文件夹内
2、使用记事本打开config.cfg文件
3、将target_folder更改为电脑中存放pdf的文件夹
4、将save_folder更改为电脑中存放识别结果的word文件的文件夹
5、点击运行pdf2word.exe，出现“请按任意键继续”即转换完成，可按任意按键或关闭命令行界面退出。

# ************************** #

# 重要提示 #
需要安装java环境并添加入path
请将路径设置为英文路径
tika文件夹内的.md5 和 .jar需要一起都复制进去，不能只复制一个！

# 版本特点 #
1、可以在config文件中指定关键词，也可以在是否需要重新制定关键词中选择 1：需要 来进行重新指定
2、browsing 模式下，会监听target_folder文件夹，如果有新增文件，则会对新增文件进行实时处理，处理选择的文段会显示在窗口界面，同时会存为同名的word
3、batch模式下，会将target_folder文件夹中的文件全部处理，同时会存为同名的word
