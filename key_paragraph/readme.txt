# ************************** #
0、安装java并配置环境，运行关闭快速编辑.bat
1、将tika文件夹下的两个文件复制粘贴到C:\Users\用户名\AppData\Local\Temp文件夹内
2、使用记事本打开config.cfg文件
3、将target_folder更改为电脑中存放pdf的文件夹
4、将save_folder更改为电脑中存放识别结果的word文件的文件夹
5、点击运行pdf2word.exe，出现“请按任意键继续”即转换完成，可按任意按键或关闭命令行界面退出。

# ************************** #

# 重要提示 #
需要安装java环境并添加入path
请将路径设置为英文路径
tika文件夹内的.md5 和 .jar需要一起都复制进去，不能只复制一个！

# 建议选择 #
   1：表示保留结尾符号和大写开头单词结尾换行;
   （如果是英文）处理空格问题
   （如果是中文）不处理空格问题
   0：不适用正则


# 出现闪退 #
请检车路径是否设置正确，存放pdf和word的文件夹是否均已创建，且名称与config文件中对应

# 版本特点 #
1、新增行处理，其中包括选项：
    1：表示保留结尾符号和大写开头单词结尾换行;
    2：表示仅保留结尾符号结尾换行;
    0：表示不处理。
2、新增空格处理，处理 a b c d e f g h 的问题
3、新增页眉页尾处理：
    1：匹配仅包含全大写单词和空格的一行（eg：THE MATING MIND ）；
    2：匹配包含全大写单词,数字和空格的一行（eg：12 THE MATING MIND）；
    3：匹配不包含小写字母的一行（eg：F&  #IRSTANCHORB&OOKSEDITION , APRIL 2001 ）;
    0：手动输入正则表达式。
